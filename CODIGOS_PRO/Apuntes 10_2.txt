Elementos -> Node
	Node final -> Button/ Spinner / ComboBox / ChoiceBox / RadioButton / Textfield / Text / List / Table
	Node container -> estructuran internamente otros elementos. Como se disponen los elementos finales -> layouts
		HBox -> los elementos se disponen horizontalmente
		VBox -> los elementos se disponen verticalmente
		BorderPane -> los elementos se disponen mediante puntos cardinales (izq, der, arr, abj, centro)
		GridPane -> los elementos se disponen en "tabla"
		FlowPane -> los elementos se disponen desde el centro repartiendo el espacio

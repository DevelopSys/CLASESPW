Esto es un ejemplo de lectura de ficheros
Esto es otra linea en el fichero
Los flujos de ficheros:
    Entrantes: lecturas -> obtengo el contenido
    Salientes: escrituras -> saco el contenido
    File -> carpeta (directory / file)
        si es fichero -> createNewFile()
        si es carpeta -> mkdir o mkdirs

Texto plano -> .txt -> ASCII
Todos los flujos deben cerrarse
Lecturas
    File -> (ponlo en modo lectura) FileReader - caracter a caracter -> BufferedReader (lectura "grande") - linea a linea

Escritura
    File -> (ponlo en modeo escritura) FileWriter - caracter a caracter -> BufferedWriter (puedo poner un salto de linea)
